# README #

A Discord bot used to bring life to your servers through an RPG-styled in-chat game.

### Current State ###

The project is still being worked on. Progress is slow but steady. Current functionality:
	* Creation of a new game environment uppon joining a new server
	* Maintenance of existing game environments uppon startup

### Adding The Bot To Your Server ###

Any server owner can add the bot to their server using the following link:
https://discordapp.com/oauth2/authorize?client_id=487744442531315712&scope=bot&permissions=1342590032
